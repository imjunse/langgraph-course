# LangGraph AI 에이전트 개발 강의 정리

> **https://imjunse.github.io/langgraph-course/**

LangGraph로 AI 에이전트를 개발하는 9시간+ 강의의 핵심 내용을 정리한 인터랙티브 웹페이지입니다.

## 내용

- **LangGraph 개요** - 저수준 오케스트레이션 프레임워크 소개
- **4가지 핵심 개념** - State, Node, Edge, Conditional Edge
- **5강 커리큘럼** (클릭하여 상세 내용 확인)
  - CH 01: 오리엔테이션 & 환경 설정
  - CH 02: LangGraph 기초 (State/Node/Edge, 에이전트 구축 5단계, 코드 예시)
  - CH 03: 에이전트 아키텍처 (Orchestration & Worker, Evaluator & Optimizer)
  - CH 04: 메모리 & 타임트래블 (Checkpoint, Time Travel, Forking)
  - CH 05: 멀티 에이전트 시스템 (Dynamic Worker, Dynamic Interrupt)
- **LangChain vs LangGraph 비교**
- **환경 설정 가이드**

## 사용법

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없이 단일 HTML 파일로 동작합니다.

```bash
open index.html
```

## 기술

- Single-file HTML (CSS/JS inline)
- Dark theme, scroll animations
- Accordion UI for chapter details
- No external dependencies
